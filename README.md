# CS4463 Steganography Project

This repository contains all of the files for our Summer 2026 CS4463 Steganography project.

Our goal is to build a Python program that can hide a message inside a 24-bit BMP image and then extract that message back out without noticeably changing the image. Everything we use throughout the project is organized here so everyone on the team has access to the same files.

## What's in this repository?

- **code** – All of the Python files used to build the program.
- **test_files** – Images, message files, and anything we use to test our program.
- **report** – Our milestone and final reports.
- **references** – Research papers and other sources used for the project.
- **diagrams** – Flowcharts and diagrams that help explain how our program works.
- **docs** – Notes, meeting information, setup instructions, and anything else that helps us stay organized.

As we continue working on the project, this repository will be updated with our code, reports, testing results, and any other files we create.
